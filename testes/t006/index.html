As tags HTML:

<p>: para paragrafos

<br>: para quebras de linhas

<hr>: para uma linha horisontal

<img>: para imagens

<h1, h2, h3, h4, h5 e h6>: para titulos

<link:favicon>: dentro da tag <head> para adicionar uma favicon

<strong>: para destaque(negrito com semantica)

<b>: para negrito (sem semântica)

<i>: para texto em italico(sem semantica)

<em>: para texto com enfase(italico com semantica)

<mark>: para marcar o texto

<big>: para um texto maior (não usar por que foi removida do HTML5 por nao ter semantica)

<small>: para um texto menor (tem semantica)

<del>: para um texto deletado

<ins>: para um texto inserido (sublinhado com semantica)

<u>: para texto sublinhado(sem semantica)

<code>: para algum fragmento de codigo

<pre>: mantem a texto digitado da mesma forma que é no HTML

<q>: para citações simples

<blockquote>:para um bloco de citações

<abbr>: para Abreviações

<ol>: para listas com ordem

<ul>: para listas sem ordem

<li>: conteudo da lista

<dl>: para listas com definição

<dt>: elemento de uma lista de definição vem antes do dd

<dd>: definição do conteudo da tag dt

<a>: para hiperlink (serve para a navegação em paginas Web e download)

<video>: para colocar videos locais

<main>: para definir o conteudo pricipal do <body>

<source>: para especificar múltiplos recursos de mídia de elementos <picture><audio><video>

<picture>: é um container usado para especificar múltiplos elementos para um elemento específico <img> contido nele

Texto marcado

Podemos criar tambem um texto marcado usando a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo. Com a tag BIG (nao usar por nao ter uma semantica) e a tag SMALL(tem semantica)
